# production-grade-api-gateway

Monorepo foundation for a production-grade API Gateway architecture.

## Structure

- `services/` — Microservices (api-gateway, user-service, order-service, payment-service)
- `packages/` — Shared libraries
- `infra/` — Docker Compose, nginx, and Redis infrastructure

## Prerequisites

- Node.js 22+
- pnpm 9+

## Workspace

```bash
pnpm install
```
